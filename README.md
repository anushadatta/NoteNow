# NoteNow

NoteNow is a native mobile application build with _Flutter_. It is equipped with login, registration, and password reset capabilities, which have been implemented with _FirebaseAuth_. Moreover, it allows the user to create & edit notes & alarms for important tasks. All this data is stored in the no SQL cloud database provided by _Cloud Firestore_.

## Features

### User Account
* Login
* Registration
* Forgot Password

### Notes & Alarms
* Functionality of adding/editing/deleting Notes/Alarms. 
* Input validation for the Create Note/Alarm form.
* Radio button to mark completion/in-completion of Note/Alarm to move it to and from History.
* Confirmation dialog box for deletion of tasks. 
* Push notifications for Alarms.
